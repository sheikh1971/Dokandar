# **App Name**: DokanHishab

## Core Features:

- Bilingual Seller Interface: A mobile-responsive UI in Bengali for staff to log inventory, daily sales, and store expenses easily.
- Real-time Firebase Ledger: Cloud-synced database to store inventory, sales figures, and cash reports directly using real-time Firebase listeners.
- End-of-Day Cash Reconciliation: Calculates expected cash based on sales minus expenses and logs discrepancies between expected and physical cash.
- Owner’s Financial Dashboard: A desktop-optimized command center for tracking revenue, COGS, operating expenses, and net profit in real-time.
- AI Profit Insights Tool: Uses a tool to analyze weekly sales patterns and expenses, providing a natural language summary of profit trends and potential leakages.
- Stock Audit Tracker: Automatically calculates live stock counts based on purchases added and sales entries reported.
- Fraud Detection Log: Aggregates cash mismatch data and reports anomalies to the owner for business accountability.

## Style Guidelines:

- High-energy Neon Palette: Vibrant Electric Green (#00FF41) for sales, Hot Magenta (#FF00FF) for expenses, and Electric Cyan (#00E5FF) for inventory highlights.
- Background: Deep Midnight Black (#050505) with multi-color neon glassmorphism effects and glowing borders.
- High-Contrast Accents: Warning Orange (#FF9100) and Laser Lemon (#FAFF00) used for alerts and discrepancy indicators.
- Headlines in 'Orbitron' for a futuristic data-heavy feel, with body text in 'Montserrat' for readability amidst intense colors.
- Multicolor 3D-style icons with neon glow effects to distinguish between different transaction types.
- Hyper-dense, color-coded grid blocks where every section uses a distinct border color to separate financial categories.
- Rainbow-pulse borders for active input fields and high-velocity sliding transitions between the seller and owner views.
