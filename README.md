# Dokandar

This is a NextJS starter in Firebase Studio.

## Repository
[https://github.com/sheikh1971/Dokandar.git](https://github.com/sheikh1971/Dokandar.git)

### Project Overview
Dokandar is a smart business management ledger built with Next.js, Firebase, and Genkit. It features:
- **Dual-role access**: Secure portals for Super Admins (Owners) and Sellers (Staff).
- **Real-time POS**: Interactive cart management for daily sales tracking.
- **Dynamic Overheads**: Manage fixed costs like Salary and Rent with autonomous monthly logging.
- **Daily Break-even Threshold**: Real-time sales targets calculated to cover mandatory expenses.
- **AI Financial Forensics**: Natural language summaries of profit trends and financial health.

To get started, take a look at `src/app/page.tsx`.
